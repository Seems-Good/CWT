## Coach's Whistle Tracker (CWT) - Simple addon to alert player when Emerald Coach's Whistle is missing

### Features:
- Reminds user with popup text on screen to put coaches whistle buff out.
- Only works when trinket is in slot 13/14.
- Only checks outside of combat and always hides the reminder upon entering combat

**Usage:**
```
/cwt    - Opens Options/Settings Menu
```
